# Doc Site Portfolio Starter

Build a professional technical writing portfolio site using MkDocs and AI-assisted content generation.

## Overview

This project provides a template and workflow for technical writers to create their own portfolio documentation site, collaborating with AI to draft and refine content.

## What's Included

- **Template**: Pre-configured MkDocs site structure
- **Scripts**: AI-assisted content generation tools
- **Prompts**: Reusable prompts for portfolio content

## Quick Start

See the [full tutorial](../docs/doc-site/tutorial/index.md) for step-by-step instructions.

## Prerequisites

- Python 3.8+
- AI API key (Anthropic Claude or OpenAI GPT)
- GitHub account (for deployment)
