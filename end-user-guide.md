# Expense Reporting Guide

!!! note "Context"
    **Audience:** Non-technical employees submitting expense reports through a SaaS finance tool. No prior system knowledge assumed.  
    **Goal:** Let users complete common tasks independently without raising a support ticket.  
    **Constraints:** Had to be clear enough for occasional users who might open the docs weeks apart.

---

## Submit an expense report

Reports submitted before the 25th of the month are included in that month's payroll cycle.

1. From the dashboard, select **Expenses → New Report**.

2. Give the report a name that identifies the trip or project — for example, *Berlin offsite – March 2025*.

3. Click **Add expense** for each line item and fill in:
    - Amount and currency
    - Date of the expense
    - Category (Travel, Meals, Equipment, etc.)
    - A brief description

4. Attach a photo or scan of the receipt. Accepted formats: JPEG, PNG, PDF.

5. Review the total, then click **Submit for approval**.

You'll receive an email confirmation. Your manager has 5 business days to approve or return the report.

!!! tip "Missing a receipt?"
    You can submit a missing receipt declaration for expenses under £25. Select **No receipt available** on the line item and enter a reason.

---

## Check approval status

Go to **Expenses → My Reports** to see the current status of any submitted report.

| Status | Meaning |
|--------|---------|
| **Pending** | Submitted and awaiting manager review |
| **Approved** | Approved — payment will be included in the next payroll run |
| **Returned** | Returned for correction — check the comments for details |

---

## Edit or resubmit a returned report

If a report is returned, you'll receive an email with your manager's comments.

1. Go to **Expenses → My Reports** and open the returned report.
2. Make the requested changes — update line items, attach missing receipts, or correct descriptions.
3. Click **Resubmit for approval**.

!!! warning
    You cannot edit a report while it is **Pending** or **Approved**. Contact your manager to have it returned to you if changes are needed.
